# RedFox 서비스 기능 정리

---

## 🏢 사장 (Owner)

### 주요 기능
| 기능 | 설명 |
|------|------|
| **대시보드** | 오늘 예상 매출, 예약 수, 근무 매니저, 운영 가게 현황을 한눈에 |
| **가게 관리** | 여러 가게 등록/수정/삭제, 실장 초대 및 관리 |
| **매출 통계** | 가게별 상세 분석 - 7일간 매출 추이, 메뉴/매니저별 실적 |
| **실장 초대** | 이메일/이름 검색으로 실장 추가, 초대 링크 생성 |

### 장점
- **다중 가게 통합 관리**: 여러 지점을 하나의 계정에서 관리
- **실시간 현황 파악**: 30초마다 갱신되는 대시보드로 현장 상황 즉시 파악
- **데이터 기반 경영**: 메뉴별/매니저별 실적 분석으로 인기 서비스와 우수 인력 파악
- **권한 위임**: 실장에게 일상 운영을 맡기고 사장은 전략적 관리에 집중

---

## 👔 실장 (Admin/Staff)

### 주요 기능
| 기능 | 설명 |
|------|------|
| **예약 관리** | 대기/확정/취소 예약 처리, 손님 평점 확인, 메시지 전송 |
| **매니저 관리** | 소속 매니저 목록, 가상 매니저 등록, 채팅 |
| **출근 관리** | 매니저 스케줄 승인/거절, 출근 요청 발송 |
| **가용 매니저 찾기** | 날짜/시간별 가능한 매니저 검색 후 출근 요청 |
| **가입 요청 관리** | 가게에 지원한 매니저 승인/거절 |
| **고객 차단** | 문제 고객 차단 기능 |

### 장점
- **원클릭 스케줄 관리**: 매니저 출근표 승인/거절이 간편
- **인력 부족 즉시 해결**: 가용 매니저 검색으로 빈 자리 빠르게 채움
- **고객 관리**: 평점 확인으로 단골/블랙리스트 구분, 직접 메시지 전송
- **가상 매니저 시스템**: 시스템에 가입하지 않은 매니저도 가상으로 등록 가능
- **효율적 커뮤니케이션**: 매니저/손님과 앱 내 채팅

---

## 💆 매니저 (Manager)

### 주요 기능
| 기능 | 설명 |
|------|------|
| **내 스케줄** | 전체 가게 통합 일정 확인 |
| **출근 등록** | 원하는 날짜/시간에 출근 가능 신청 |
| **내 예약** | 배정된 예약 목록, 메뉴/가격/손님 정보 확인 |
| **가게 탐색** | 새 가게 검색, 지원하기 |
| **출근 요청** | 실장이 보낸 출근 요청 수락/거절 |
| **프로필 관리** | 사진, 소개, 전문 분야, 신체 정보 등록 |
| **에이전시 탐색** | 에이전시에 소속되어 추가 일감 확보 |

### 장점
- **자유로운 근무**: 여러 가게에 소속되어 원하는 시간에 출근 등록
- **수요 파악**: 가게별 오늘 예약 수와 주간 평균 비교로 수요 높은 곳 확인
- **투명한 예약 정보**: 배정된 손님, 메뉴, 가격 미리 확인
- **프로필 홍보**: 전문 분야, 사진 등록으로 손님에게 어필
- **유연한 일정 관리**: 본인이 가능한 시간만 등록, 요청이 오면 수락/거절 선택

## 👤 손님 (Customer)

### 주요 기능
| 기능 | 설명 |
|------|------|
| **가게/매니저 탐색** | 가게 목록, 매니저 검색 및 상세 정보 확인 |
| **예약하기** | 원하는 가게/매니저/메뉴/시간 선택하여 예약 |
| **내 예약** | 예약 현황 확인, 상태 추적 (대기→확정→완료) |
| **즐겨찾기** | 자주 가는 가게, 선호 매니저 저장 |
| **포인트** | 적립/사용, 다른 사용자에게 선물 |
| **평점 남기기** | 서비스 완료 후 별점/리뷰 작성 |
| **메시지** | 실장/매니저와 직접 채팅 |

### 장점
- **쉬운 예약**: 가게나 매니저 중심으로 예약 가능
- **투명한 정보**: 매니저 프로필, 전문 분야, 평점 미리 확인
- **실시간 소통**: 예약 전/후 채팅으로 문의 가능
- **즐겨찾기**: 단골 가게/매니저 빠른 재예약
- **포인트 적립**: 이용할수록 혜택 누적

---

## 🔧 서비스 관리자 (Superadmin)

### 주요 기능
- 전체 사용자/가게/예약 통합 관리
- 역할별 사용자 현황, 가게 유형 관리
- 사장 가입 승인, 비밀번호 초기화
- 역할/매니저 노출 설정

---

## 🏬 에이전시 (Agency)

### 주요 기능
- 소속 매니저 관리
- 가게에 매니저 파견 (Dispatch)
- 파견 현황 추적

### 장점
- **중개 역할**: 매니저를 가게에 연결해주는 플랫폼
- **파견 시스템**: 체계적인 인력 파견 관리

---

## 핵심 특징 요약

| 특징 | 설명 |
|------|------|
| **역할 기반 시스템** | 사장→실장→매니저→손님 계층 구조 |
| **유연한 인력 관리** | 매니저가 여러 가게에서 일할 수 있는 긱 이코노미 모델 |
| **실시간 소통** | 모든 역할 간 채팅 기능 |
| **평점 시스템** | 손님/매니저 모두 평점 관리로 신뢰 구축 |
| **데이터 분석** | 매출, 예약, 인기 메뉴/매니저 통계 |
